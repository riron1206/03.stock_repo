# pair_trade
株価csvからペアトレードに向いている銘柄を探索
- 実行するには株式投資メモ<<https://kabuoji3.com/>>の株価csvファイルが必要
- 2銘柄の累積収益率の残差グラフに周期性があるかを確認する必要があるためnotebookで実行すべき
- 参考:run_pair_trade.ipynb
